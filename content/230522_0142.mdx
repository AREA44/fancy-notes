# Git ‚Äì Tips & Tricks

## üÜï Start a New Git Repository

Initialize and push a new Git repository:

```bash copy
git init                        # Initialize a new repo
git add .                       # Stage all files
git commit -m "first commit"    # Initial commit
git branch -M main              # Rename branch to main
git remote add origin https://github.com/OWNER/REPOSITORY
git push -u origin main         # Push to remote main
````

## üîó Change the Remote Repository URL

Update an existing remote URL:

```bash copy

```

```bash copy

```

## üß¨ Clone All GitHub Repositories (User or Org)

Works on **Windows**, **macOS**, or **Linux** using **Git Bash**:

```bash copy
CNTX={users|orgs}; PAGE=1
curl "https://api.github.com/$CNTX/OWNER/repos?page=$PAGE&per_page=100" |
  grep -e 'clone_url' |
  cut -d \" -f 4 |
  xargs -L1 git clone
```

<Callout type="info">
  Set <code>CNTX=users</code> to download your personal repositories, or <code>CNTX=orgs</code> for your organization‚Äôs repositories.
</Callout>

<Callout type="warning">
  GitHub API limits results to 100 per page. Increase the <code>PAGE</code> variable to fetch all repositories across multiple pages.
</Callout>

## üóëÔ∏è Delete a Branch in Git

### Delete a **local** branch:

```bash copy
git branch -d branch_name      # Safe delete
git branch -D branch_name      # Force delete (use with caution)
```

### Delete a **remote** branch:

```bash copy
git push origin --delete branch_name
```

This command tells Git to delete the branch from the remote repository (like GitHub or GitLab).
