## ðŸŽµ Download an MP3 from YouTube

To extract audio in MP3 format from a YouTube video:

```bash copy
yt-dlp --extract-audio --audio-format mp3 "<YouTube video URL>"
```

<Callout type="tip">
  Replace <code>&lt;YouTube video URL&gt;</code> with the actual URL of the video you want to download.
</Callout>

<Callout type="info">
  The resulting MP3 will be saved in your current working directory unless a different output path is specified using the <code>-o</code> flag.
</Callout>
